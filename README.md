# CauArchives 🎧💿🔊

Este repositório apresenta o código-fonte do **CauArchives**, uma galeria interativa e imersiva de álbuns musicais com física 3D em tempo real. Mais do que um simples player, o objetivo deste projeto foi unir minhas paixões por **Música, UX/UI Design e desenvolvimento Front-End**, criando um acervo pessoal e uma experiência interativa nostálgica.

A proposta foca em fugir do formato de lista tradicional, recriando no navegador a sensação tátil de procurar discos físicos em uma prateleira, com uma interface minimalista e fluida.

### Objetivos e Habilidades Demonstradas

Este projeto foi desenvolvido para evidenciar minha capacidade de criar interfaces complexas, responsivas e animadas, com foco em:

* **UX/UI Design Imersivo:** Planejamento de uma experiência tátil focada no *crate digging* (procurar discos), alinhada a uma estética nostálgica dos anos 2000 (inspirada no minimalismo da Apple e nos menus do Nintendo Wii).
* **Animações e Física 3D:** Uso avançado da biblioteca GSAP (GreenSock) e ScrollTrigger para manipular o DOM no espaço 3D (eixo Z), garantindo fluidez a 60 frames por segundo.
* **Desenvolvimento de Áudio Customizado:** Construção de um player de música nativo via JavaScript (sem o uso de iframes de terceiros), controlando fade de volume, play/pause e integração de prévias das faixas.
* **Acessibilidade e Responsividade:** Implementação de navegação mista (Scroll e Setas do Teclado no Desktop; Swipe e limites de quebra de texto no Mobile), além de animações de preloader cinemático e interações de *Tilt 3D* com o mouse.

-----

### Tecnologias Utilizadas

* **HTML5** (Estruturação semântica e manipulação de espaço 3D real)
* **CSS3 & Tailwind CSS** (Estilização ágil, responsiva e design system limpo)
* **JavaScript (ES6+)** (Lógica do player de áudio, Web Audio API e eventos de interface)
* **GSAP / GreenSock** (Core das animações fluidas e cálculo de profundidade 3D)
